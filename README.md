## 📝 教程内容

本教程包含以下详细步骤：

- ✅ Windows 安装（PowerShell 管理员 + 一键安装脚本）

- ✅ 初始化向导 7 步详解
  - 模式选择（QuickStart vs Advanced）

OpenClaw 是一个自托管的多频道 AI 网关，可以将 WhatsApp、Telegram、Discord、iMessage、飞书等聊天应用连接到你的 AI 助理。

- 官网：https://openclaw.ai
- 文档：https://docs.openclaw.ai
