# 微信 GUI 控制 Skill（给熊叔）

这个 skill 用于在 macOS 上通过 GUI 自动化控制微信桌面端，重点解决群聊中**真实 @ 成员**的问题。

## 核心能力

- 定位微信群并聚焦输入框
- 通过 `@` 候选列表点选成员（非手打）
- 发送前文案确认
- 发送前后校验，避免“看起来 @ 了但实际没提及”

## 关键规则

1. 默认**预演模式**：只做到草稿和 @ 校验，不直接发送。
2. 只有用户明确回复“发”，才执行真实发送。
3. 未确认不重复发送。

## 文件结构

- `SKILL.md`：主流程与硬规则
- `references/send-checklist.md`：发送检查清单

## 适用场景

- “去某个微信群发一条消息并 @ 指定两人”
- “发之前先确认文案，再发送”
- “发送后确认是否真实 @ 成功”
