# WeChat 发送检查清单（群聊@）

默认模式：预演（只做到草稿与@校验，不发送）。
仅当用户在当前轮明确回复“发”时，才允许实际发送。

发送前必须全部满足：

- [ ] 已锁定目标群名称（避免串群）
- [ ] 输入框焦点正确
- [ ] 已通过候选列表点选 @成员1
- [ ] 已通过候选列表点选 @成员2
- [ ] 草稿中可见两位成员名（不是只有 `@ @`）
- [ ] 正文文案已确认
- [ ] 用户明确回复“发”

发送后：

- [ ] 复核聊天列表/会话预览出现该条消息
- [ ] 向用户回报 done/blocked
- [ ] 若用户反馈“未实际@到”，立即回滚到“发前确认”阶段重做
