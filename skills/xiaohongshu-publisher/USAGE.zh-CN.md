# USAGE（小红书全自动发布）

下面是 3 条可直接复制的调用方式。

## 1) 从 0 到草稿

```text
用 xiaohongshu-publisher 帮我做一篇小红书草稿。
主题：OpenClaw 自动化发布
目标人群：独立开发者
风格：实操干货
输出：标题 + 正文 + 配图顺序建议
```

## 2) 进入“预演不发送”

```text
用 xiaohongshu-publisher 进入发布预演模式：
- 先上传图片并填好标题正文
- 但不要点击发布
- 把最终发布内容发我确认
```

## 3) 用户确认后再发送

```text
确认发布。
按刚才预演内容执行，并在发布后去管理页核验状态，再给我闭环汇报。
```

---

## 推荐口令（简版）

- `先出草稿，不发`  
- `进入预演，等我确认`  
- `确认发布`

## 注意

- 未收到明确“确认发布/发吧/可以发”，skill 不应点击发布。
- 发送后必须回报：当前状态（审核中/已发布）+ 下一步复查时间。
